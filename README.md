## ⚡ The Polymorphic Design Protocol

This portfolio isn't just a static site—it's a multi-state architectural demonstration. Built with a custom design system that allows users to toggle between three distinct "Design Protocols" in real-time:

### 1. 💎 MINIMAL (The Professional)
*   **Aesthetic**: Clean, elite, and whitespace-heavy.
*   **Persona**: Enterprise-ready engineering and executive-level clarity.
*   **Typography**: *Outfit* & *Inter*.

### 2. 🎨 STREET (The Latest Era)
*   **Aesthetic**: Messy, funky, and anti-design.
*   **Persona**: Experimental urban energy with neon glows, glitch effects, and spray-paint stencils.
*   **Typography**: *Bungee* & *Syne*.

### 3. 🧱 BRUTAL (The Industrial)
*   **Aesthetic**: Raw, high-contrast, and no-nonsense.
*   **Persona**: Pure engineering rigor with flat shadows and technical specification vibes.
*   **Typography**: *Unbounded* & *JetBrains Mono*.

## 🚀 Technical Core

- **Framework**: [Vue 3](https://vuejs.org/) with `<script setup>` for optimized reactivity.
- **Build System**: [Vite](https://vitejs.dev/) for near-instant HMR and performance.
- **State Management**: [Pinia](https://pinia.vuejs.org/) (Custom `useThemeStore` for multi-style orchestration).
- **Animation Engine**: [GSAP](https://greensock.com/gsap/) + [ScrollTrigger](https://greensock.com/scrolltrigger/) for high-precision scrolling interactions and reveal sequences.
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) + Custom CSS variables for the polymorphic theme engine.
- **Iconography**: [Lucide-Vue-Next](https://lucide.dev/) for crisp, scalable vector icons.

## 🚀 V2 Update Complete

The architecture has been upgraded to **Version 2.0** with massive performance optimizations and visual enhancements:
- **Mobile Performance Paradigm**: Dynamic loading and selective pausing of heavy 3D canvases and complex glitch effects (FuzzyText, Polymorphic Canvas) exclusively on touch devices to ensure silky-smooth performance and preserve battery life.
- **Adaptive Typography & Layouts**: Re-engineered floating metric cards and responsive text grids that intelligently adapt from complex stacked layouts on desktop to clean, readable single-row layouts on mobile screens.
- **Enhanced Visual Context**: The hero section now features a frosted hollow-ring ambient backlight and expanded orbital text (`BUG SLAYER CERTIFIED * 100% CHAD VERIFIED * ELITE CODE ARCHITECT *`), offering a richer, premium feel.
- **Global App Navigation**: Added a smart, scroll-aware floating Action Button that instantly returns users to the primary viewport, auto-hiding near page boundaries.
